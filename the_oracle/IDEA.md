Es un ejercicio de fundamentos de backend.

Enunciado:
1. Construir un proyecto backend llamado THE ORACLE, que consume un servicio de I.A. de Gemini.
2. Integraremos el framework Express JS con la estructura que vimos hasta ahora.
3. El endpoint se llamará /generate-content con el método POST y un cuerpo que envía una pregunta.
4. Consumiremos una función asincrónica que interactúa con Gemini API y retorna la respuesta.
5. Debemos manejar cualquier tipo de error mediante la implementación de try-catch.
6. Debemos controlar errores de la API de Gemini, de manera personalizada, aplicando los Estados
propios de una API REST.
7. No debemos revelar los códigos de error de Gemini API.
8. Ten presente crear el archivo .env para la definición de la clave de la Gemini API Key.
